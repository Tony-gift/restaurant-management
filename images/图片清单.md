# 图片资源清单

为了让新的登录界面正常显示，您需要在 `images/` 目录下放置以下图片：

## 必需图片文件

### 1. logo.png
- **用途**: 登录页面顶部Logo
- **建议尺寸**: 200x200 像素
- **格式**: PNG (支持透明背景)
- **位置**: `/images/logo.png`

### 2. employee.png
- **用途**: 员工身份选择卡片图标
- **建议尺寸**: 64x64 像素
- **格式**: PNG
- **位置**: `/images/employee.png`
- **设计建议**: 可以使用办公室、工作人员相关的图标

### 3. franchise.png
- **用途**: 加盟商身份选择卡片图标
- **建议尺寸**: 64x64 像素
- **格式**: PNG
- **位置**: `/images/franchise.png`
- **设计建议**: 可以使用商店、合作、握手相关的图标

### 4. recruit.png
- **用途**: 求职者身份选择卡片图标
- **建议尺寸**: 64x64 像素
- **格式**: PNG
- **位置**: `/images/recruit.png`
- **设计建议**: 可以使用简历、求职、面试相关的图标

## 界面效果说明

新的登录界面分为两个阶段：

### 阶段一：身份选择
- 显示公司Logo
- 显示三个身份选择卡片（员工、加盟商、求职者）
- 每个卡片包含图标、标题和描述

### 阶段二：登录表单
- 显示选中的身份类型
- 提供"切换身份"按钮返回选择界面
- 包含用户名、密码输入框和登录按钮

## 图片准备建议

如果您暂时没有合适的图片，可以：
1. 使用在线图标库（如 Iconfont、Flaticon）下载免费图标
2. 使用简单的文字图标临时替代
3. 使用颜色块作为占位符

## 临时解决方案

如果图片暂时无法准备，界面仍然可以正常工作，只是图标位置会显示默认的图片占位符。
