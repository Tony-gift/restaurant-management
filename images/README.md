# 图片资源放置说明

## 图片目录结构
```
cl/
├── images/
│   ├── logo.png          # 登录页面Logo图片
│   ├── icons/            # 图标目录
│   ├── backgrounds/      # 背景图片目录
│   └── avatars/          # 头像图片目录
```

## 如何添加Logo图片

1. 将您的Logo图片文件重命名为 `logo.png`
2. 将图片文件放置到 `cl/images/` 目录下
3. 建议图片尺寸为 200x200 像素或更高（系统会自动缩放）
4. 支持的格式：PNG、JPG、JPEG

## 图片命名规范

- 使用小写字母和连字符
- 避免使用空格和特殊字符
- 示例：`company-logo.png`, `user-avatar.jpg`

## 图片优化建议

- 压缩图片以减少小程序包大小
- 使用适当的图片格式（PNG用于透明背景，JPG用于照片）
- 图片大小建议控制在100KB以内
