# 企业管理系统微信小程序

这是一个企业管理系统的微信小程序前端项目，包含员工管理、加盟管理和入职申请三个主要模块。

## 功能特点

### 登录界面
- 三种用户类型选择：员工、加盟、入职
- 统一的登录表单界面
- 用户类型区分和权限管理

### 注册功能
- 支持加盟和入职用户注册
- 简单的用户名和密码注册方式
- 表单验证和错误提示

### 员工功能模块
- **考勤管理**：上下班打卡、考勤统计、打卡记录
- **请假申请**：请假类型选择、申请流程、状态跟踪
- **工资查询**：工资明细、历史记录、年度统计
- **个人档案**：基本信息、工作信息、教育背景

### 加盟功能模块
- **加盟申请**：详细的加盟申请表单
- **申请进度**：实时跟踪审核进度和状态

### 入职功能模块
- **入职申请**：完整的个人信息填写
- **申请状态**：查看审核进度和面试安排

## 技术实现

- 使用微信小程序原生框架开发
- 采用模块化的页面结构设计
- 实现了完整的用户状态管理
- 包含丰富的表单验证和交互反馈
- 响应式设计，适配不同屏幕尺寸

## 项目结构

```
├── app.js              # 应用入口文件
├── app.json            # 应用配置文件
├── app.wxss            # 全局样式文件
├── sitemap.json        # 站点地图配置
├── pages/              # 页面目录
│   ├── login/          # 登录模块
│   ├── register/       # 注册模块
│   ├── employee/       # 员工模块
│   ├── franchise/      # 加盟模块
│   └── recruit/        # 入职模块
└── images/             # 图片资源目录
```

## 使用说明

1. 使用微信开发者工具打开项目
2. 在登录页面选择用户类型
3. 根据接口服务实际返回的用户信息完成登录
4. 根据用户类型进入相应的功能模块

## 注意事项

- 默认接口地址在 `utils/api.js` 中配置，请根据部署环境修改 `BASE_URL`
- 登录、注册、加盟申请、入职申请和员工打卡等功能已接入后端 API
- 图片资源需要替换为实际的业务图标
